什么是白帽子，黑帽子，红帽子黑客? 不同类型的黑客解释

发表于2022-09-23|更新于2022-09-23
|阅读量:

你是否听说过“白帽黑客”或“黑帽黑客”这个词，并且想知道它的意思？
在本文中，你将通过将黑客与多多少少代表他们的漫威或 DC 英雄进行比较，了解他们是如何被分类的，以及他们是做什么的。

##黑客是什么？

黑客是利用自己的技能攻破网络安全防御的个人。在网络安全领域，黑客通常被一个“帽子”系统分类。
这种制度很可能来自古老的牛仔电影文化，在那里，好的角色通常戴白帽子，坏的角色戴黑帽子。
网络空间的主要职责有三:
1.White Hats 白帽子
2.Grey Hats 灰帽子
3.Black Hats 黑帽子

然而，随着时间的推移，也出现了一些其他的情况，例如:
1.Green Hats 绿帽子
2.Blue Hats 蓝帽子
3.Red Hats 红帽子

让我们深入了解一下这些不同类型的黑客都做些什么，好吗？

##1.白帽黑客
白帽子就像漫威的美国队长。不管是什么时候，什么年龄，他们总是坚持正义，
通过在黑帽子之前发现和报告系统漏洞来保护平民和组织。他们通常为组织工作，
担任网络安全工程师、渗透测试员、安全分析师、 CISO (首席信息安全官)和其他安全职位。

在这些组织下，他们执行下列任务:
Scanning networks 扫描网络
Configuring IDSs (Intrusion Detection Systems) 配置入侵检测系统
Ethically hacking computers to find vulnerabilities and report them so they can be addressed
合乎道德的黑进电脑，找出漏洞并报告，以便解决
Programming honeypots (Traps for the attackers 😼)
编程蜜罐(攻击者的陷阱)
Monitoring network activity for suspicious activity
监测网络活动以防止可疑活动

##2.灰帽黑客
DC 的黑暗骑士和灰帽黑客有很多共同点。他们都想为正确的事情挺身而出，但却用了一些非常规的方法。
灰帽子黑客是白帽子和黑帽子之间的平衡。与白帽黑客不同的是，他们不会要求获得黑客系统的许可，但不会执行任何其他非法活动，比如黑帽黑客。
灰帽子的历史颇具争议。这使得他们很难真正分类，特别是当他们的道德指南针有点混乱，或者他们所做的事情看起来更像是黑帽子而不是白帽子。
有些人甚至因为他们的所作所为进了监狱。但也有一些人成为了人民的英雄，政府和大机构的敌人。

灰帽黑客的一些著名例子如下:
Anonymous (World famous hacktivist group)
匿名者(世界著名的黑客组织)
HD Moore (Creator of Metasploit)
HD Moore (Metasploit 的创建者)
Adrian Lamo (aka the homeless hacker)
Adrian Lamo (又名流浪汉黑客)
Khalil Shreateh (Hacked the facebook account of Mark Zuckerburg 🤣)
Khalil Shreateh (黑了 Mark Zuckerberg 的 Facebook 账号)

##3.黑帽黑客

是时候介绍这些有害的东西了。小丑和黑帽子就像一个豆荚里的豌豆。他们为了经济利益、挑战或者仅仅是为了好玩而从事非法活动。
他们寻找在互联网上容易受到攻击的计算机，利用它们，并尽可能地利用它们。
黑帽子使用技术进入系统，就像白帽子一样。然而，他们并没有使用他们的防守技巧——相反，
他们通过做以下事情来提高他们的进攻能力:
Installing backdoors 安装后门
Maintaining access to compromised systems 维护入侵系统的权限
Performing privilege escalation 表演权限提升
Downloading private/sensitive/intellectual data 下载私人/敏感/知识资料
Installing malware such as ransomware 安装勒索软件等恶意软件
Creating phishing emails and links 创建网络钓鱼电子邮件和链接

##4.绿帽黑客
漫威女士和绿帽子简直是天造地设的一对。他们都很年轻，热情，缺乏经验，而且有冒险和从错误中吸取教训的倾向。绿帽子是黑客是新的行业，
但愿意学习成为伟大的黑客。由于现在黑客工具的可用性和易用性，绿帽子很容易陷入麻烦，因为他们可能不完全理解工具或目标的全部工作原理。
但是，他们从错误中吸取教训，积累经验。绿帽子可以升级到白色，灰色，或黑帽子黑客，因为他们继续向上移动的行列。

##5.蓝帽黑客

约翰 · 威克既不是 DC 漫画也不是漫威漫画的成员，但是炸药漫画最伟大的杀手却是所有粉丝的最爱。
威克先生和蓝帽黑客有着相同的理念: 复仇。你杀了 John Wicks 的狗，他会来找你的。你恐吓或威胁一个蓝帽子，他们也会来找你，除了你在绞刑架上的数字生活。
但是，由于我只能猜测是文化差异，蓝帽子也可能意味着外部安全专家带来测试软件的漏洞之前，它的发布。

##6.红帽黑客
我认为这个角色说明了一切。惩罚者是一个无情的反英雄，站在什么是正确的，但从来没有(我的意思是从来没有)去给罪犯第二次机会。
红帽子也是一样，他们以网络罪犯为目标，破坏一切他们能破坏的犯罪活动，永久性的。
红帽子是没人愿意招惹的黑客，即使是黑帽子也不行。其他黑客通常攻击微软的 Windows 电脑，但这些黑客，他们黑客 Linux 电脑。
他们不后悔，不再犹豫，并且通过伸张正义让黑帽子为他们的罪行付出沉重的代价。他们通过销毁目标的所有数据和备份来实现这一点，通常会使系统失效。

##结论
在这个可怕的音符下，我们已经到了这篇文章的结尾。希望你喜欢。就像我常说的，Happy hacking!
